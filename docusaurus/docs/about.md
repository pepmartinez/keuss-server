Job Queues' server accesible via STOMP and REST, built with keuss

Keuss-server provides STOMP and REST-like interfaces atop [keuss](https://pepmartinez.github.io/keuss/), plus a simple web console to check queues' statuses. It aims to offer all functionalities provided by keuss

Also, provides several prometheus metrics
